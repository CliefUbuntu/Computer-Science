# GitHub SSH Key Recovery: CliefUbuntu Account

## Reason
The original SSH key (~/.ssh/id_ed25519) was accidentally overwritten. GitHub access was broken for the `CliefUbuntu` account.

---

## Steps to Restore

1. **Generate a new SSH key**
   ```bash
   ssh-keygen -t ed25519 -C "cliefubuntu@gmail.com" -f ~/.ssh/id_ed25519_main
   ```

2. **Start ssh-agent and add the new key**
   ```bash
   eval "$(ssh-agent -s)"
   ssh-add ~/.ssh/id_ed25519_main
   ```

3. **Add the new public key to GitHub**
   - Copied using:
     ```bash
     cat ~/.ssh/id_ed25519_main.pub
     ```
   - Pasted into GitHub → Settings → SSH and GPG Keys → New SSH key

4. **Updated SSH config**
   ```ssh
   Host github-main
       HostName github.com
       User git
       IdentityFile ~/.ssh/id_ed25519_main
   ```

5. **Updated Git remote**
   ```bash
   git remote set-url origin git@github-main:CliefUbuntu/Computer-Science.git
   ```

6. **Tested the connection**
   ```bash
   ssh -T git@github-main
   ```

7. **Pulled and pushed successfully using the new key**
   ```bash
   git pull
   git add .
   git commit -m "Documented SSH key recovery steps"
   git push
   ```
